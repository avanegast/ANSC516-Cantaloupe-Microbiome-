# ANSC516-Cantaloupe-Microbiome

This repository contains the code needed to analyze the sequencing data obtain from the 
Franco-Frias et. al., 2021. 
Detailed information can be found in the published paper:

**Title:** Analysis of Bacterial Communities by 16S rRNA Gene Sequencing in a Melon-Producing Agro-environment.
**Authors:** Eduardo Franco-Frías, Victor Mercado-Guajardo, Angel Merino-Mascorro, Janeth Pérez-Garza, Norma Heredia, Juan S. León, Lee-Ann Jaykus, Jorge Dávila-Aviña & Santos García
**Published:** 2021, vol. 82, pages 613-622
**Journal:** Environmental Microbiology. 
The sequence data is stored on the NCBI Sequence Read Archive database under Bioproject PRJNA693300
(https://www.ncbi.nlm.nih.gov/sra/?term=PRJNA693300)
